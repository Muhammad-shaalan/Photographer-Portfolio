Getting Started

Welcome To Rarafolio

Created: 09/05/2017 - By: Muhammad Shaalan

Rarafolio is a Modern HTML5 portfolio & resume Template. Rarafolio is 100% responsive. This template is best suitable for Portfolio / vCard / CV / Resume template designed for creative designers, developers, freelancers, photographer or pretty much any profession around the world and in a few moments you can customize this template to suit your own needs. All files are organized and highly documented, so working with these files is a breeze.

Installing Theme

1. After unzip Theme Files, you will found HTML Folder have Theme Rarafolio this files you will upload.

2. You can view this Theme in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the Theme.
